<h1 align="center">
Full configuration guide
</h1>

Cloning the repository:-
========================
1. clone to following directory

        cd ~/.
        git clone ...

Setting up tmux:-
-------------------
1. Start installing tmux:-

        cd scripts/tmux
    check tmux README

Setting up neovim:-
-------------------
1. Start installing neovim:-

        cd scripts/neovim
    check neovim README

Edit ~/.bashrc:-
================
1. Put following at the end of ~/.bashrc

        source ~/blueback_scripts/init.sh
