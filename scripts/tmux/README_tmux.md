Building Docker image
---------------------
1. Start installing tmux:-

        ./tmux_install.sh

2. Start tmux and load:-

        tmux
        <Ctrl-a> <I>

    This will install tmux/tpm and tpm packages like
        
        tmux-powerline
        tmux-ressurect

3. Configure powerline:

        ./powerline_post_install_config.sh
